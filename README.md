# DocQuery

AI-powered document analysis platform that lets you **chat with your files**.

Upload documents such as **PDF, Word, Excel, PowerPoint, or TXT**, then ask questions, generate summaries, and extract insights instantly using AI.

🌐 Live Demo: https://docquery.online

---

## Features

### 📄 Upload Multiple Document Types
Supports common document formats:

- PDF
- DOCX
- XLSX
- PPTX
- TXT

### 💬 Chat With Your Documents
Ask natural language questions about your files and get instant AI-generated answers.

Examples:
- "Summarize this report"
- "What are the key conclusions?"
- "Extract all financial numbers"

### ⚡ Smart Summaries
Generate concise summaries of large documents within seconds.

### 🔎 Deep Document Understanding
Automatically understands:

- Tables
- Structured data
- Key points
- Context across large files

### 🚀 Fast Processing
Most documents are processed in **under 10 seconds**.

### 🔐 Privacy Focused
- Files processed securely
- Documents automatically deleted after analysis
